# Minesweeper v2.4 Responsive Touch Bars Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将首页高卡片重构为桌面横向、手机双层的单表面胶囊触控条，并收窄、减轻手机游戏顶部玻璃栏，统一升级到 2.4.0。

**Architecture:** 保留现有页面路由、游戏状态和 Liquid Glass 控制器，仅重组 `home.ts` 的展示分组并用内容驱动断点重排。游戏顶栏继续沿用原 DOM，通过 `glass-light` 材质修饰类和窄屏宽度规则减轻视觉重量；所有变化由 Vitest 静态/DOM 测试和 Playwright 几何测试锁定。

**Tech Stack:** TypeScript 6, DOM API, CSS Grid/Flexbox, Vitest, Playwright Chromium/WebKit, Vite PWA, Tauri 2.

```ts
import { expect, test } from "@playwright/test";

test("桌面首页是低矮横向胶囊且版本号在表面外", async ({ page }) => {
  await page.setViewportSize({ width: 1440, height: 900 });
  await page.goto("/");
  const panel = page.locator(".home-panel");
  const box = await panel.boundingBox();
  expect(box!.width).toBeGreaterThan(700);
  expect(box!.height).toBeLessThan(160);
  expect(box!.width / box!.height).toBeGreaterThan(4);
  expect(await panel.evaluate((el) => getComputedStyle(el).borderRadius)).toBe("999px");
  await expect(page.locator(".home > .home-ver")).toBeVisible();
});

test("390px 首页完整可触控且不横向溢出", async ({ page }) => {
  await page.setViewportSize({ width: 390, height: 844 });
  await page.goto("/");
  const buttons = page.locator(".home-panel button");
  for (let index = 0; index < await buttons.count(); index += 1) {
    const box = await buttons.nth(index).boundingBox();
    expect(box!.width).toBeGreaterThanOrEqual(44);
    expect(box!.height).toBeGreaterThanOrEqual(44);
  }
  expect(await page.evaluate(() => document.documentElement.scrollWidth)).toBe(390);
});
```
