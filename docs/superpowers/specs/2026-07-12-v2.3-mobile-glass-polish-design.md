# v2.3 Mobile Liquid Glass Polish Design

## Goal

修复 iPhone 竖屏截图中首页面板圆角不足、游戏顶栏过宽、按钮图标偏小且右偏的问题，同时保持 44px 触控目标、Liquid Glass 单层材质、键盘可达性和 reduced-motion 降级。

## Design

- 首页 `.home-panel` 继续作为唯一玻璃表面，将圆角提升到胶囊级（3rem），并略增内边距，让面板边缘在窄屏上保持连续柔和的弧线。
- 游戏顶栏不改变 DOM 语义：使用 CSS grid 让返回按钮、标题和统计操作占据稳定槽位；小屏时统计操作独立成等宽一行，避免 `margin-left:auto` 把按钮挤到右侧。
- 返回、音效、重开按钮保持最小 44px 触控区，图标字号提升；统计数字保留紧凑胶囊样式。所有布局只使用 CSS，不新增运行时状态。
- 保留现有 `prefers-reduced-motion`、`prefers-reduced-transparency`、forced-colors 和焦点样式。

## Verification

- CSS 静态断言验证圆角、grid 槽位、等宽统计操作和最小触控尺寸。
- Chromium/WebKit E2E 在 iPhone 尺寸验证首页/游戏顶栏几何关系、按钮尺寸与无控制台错误。
- 完整 Vitest、E2E、web build 和 `git diff --check` 作为交付门槛。
